# Slide in Sync Remix Runtime

Public, versioned runtime artifacts used by the Remix build of **Slide in Sync**.

The initial `v0.0.1` release contains only a harmless JavaScript probe. It exists
to verify that Remix accepts version-pinned assets served from this repository
through jsDelivr before any Defold runtime files are published here.

Production consumers must pin an immutable tag or commit. Do not use `main` or
`latest` URLs.
